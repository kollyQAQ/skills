# Originality Checklist

在输出前逐项自检：

1. 是否存在连续长句接近来源原文。
2. 是否直接复用了来源的段落结构和举例顺序。
3. 是否出现“同义词替换式改写”但论证骨架未变。
4. 是否引用了未经核实的数据或二手转述。
5. 是否把来源观点重组为自己的叙述路径。

## Red Flags

- 口吻突然切换为新闻稿或百科体。
- 术语堆叠但没有可理解的解释。
- 大段“正确但空”的套话。

## Fix Strategy

- 先写自己的结论，再补证据，不反向拼装。
- 更换论证顺序：从机制切到场景，或从场景切到机制。
- 将抽象结论改写为具体情境表达。
- 删掉无法确认的一切“精确数字”。
