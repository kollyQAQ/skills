# Answer Framework

## Short Form (快速问答)

适用于问题简单、结论明确的场景。

1. 结论（1-2 句）
2. 关键理由（2-4 点）
3. 实操建议（1-2 条）

## Standard Form (通用高赞)

适用于大多数知乎问答。

1. 开场结论
- 直接回答问题，避免铺垫过长。

2. 为什么
- 给出核心机制或因果链。
- 尽量用“现象 -> 原因 -> 影响”组织。

3. 怎么做
- 提供可执行建议。
- 建议可分轻量/进阶两档。

4. 收束
- 一句话回扣题目。
- 保留讨论空间，便于评论互动。

## Long Form (深度分析)

适用于争议大、信息量高的问题。

1. 明确问题边界
2. 提出主观点
3. 分层论证（现实层、机制层、案例层）
4. 反方观点与回应
5. 结论与行动建议

## Style Notes

- 句长交替，避免全篇长句。
- 段落控制在 3-5 行，提升移动端阅读体验。
- 有数据时优先写“结论 + 口径说明”。
